Sass (Syntactically Awesome Style Sheets) é uma linguagem de estilo para a web que é uma
extensão do CSS (Cascading Style Sheets). É usada para adicionar recursos avançados ao CSS,
como variáveis, aninhamentos, mixins, herança, entre outros,
tornando o desenvolvimento de estilos mais produtivo e fácil de manter.
Sass é compilado em CSS puro, que é interpretado pelos navegadores da web.
